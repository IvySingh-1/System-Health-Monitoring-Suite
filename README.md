# System-Health-Monitoring-Suite
System Health Monitoring Suite (macOS Edition) with Log Analyzer, ASCII Graphs &amp; Live Monitor
- Engineered a robust shell-based monitoring solution using Bash, awk, and sysctl to track 5+ key performance indicators in real-time.
- Optimized system observability by implementing a live monitor with a 2-second latency, reducing the time to detect system anomalies.
- Visualized complex performance data by developing a custom ASCII graph generator, transforming raw log entries into actionable trend analysis.
- Automated critical maintenance tasks, including threshold-based alerts for usage over 80% and timestamped system backups, ensuring zero data loss for logs and documents.
